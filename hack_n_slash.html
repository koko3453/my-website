About the Game

Hack n Slash Survivor is a fast-paced, top-down survival game inspired by Vampire Survivors. Battle endless waves of enemies, collect upgrades, and see how long you can survive!

Gameplay

  • Endless auto-attacking combat on an infinite scrolling map
  • Defeat enemies to collect XP orbs and level up
  • Choose 1 of 4 random upgrades each time you level up
  • Upgrades include Items (passive stat boosts) and Weapons (new or improved attacks)
  • Enemies become stronger and more numerous over time
  • Unleash a powerful ultimate ability (SPACE) with a 2-minute cooldown

Technical

  • Written in C using the SDL2 library for graphics and input
  • Data-driven design: game content (enemies, upgrades, etc.) defined in JSON files
  • Efficient entity and upgrade system for scalable gameplay
  • Cross-platform support (Windows, Linux, macOS)
